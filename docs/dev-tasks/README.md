## ⚠️ 关键风险控制

| 风险 | 缓解措施 |
|------|---------|
| TASK-14 改 WidgetOverlayContext 影响现有代码 | 所有新增字段为 optional，零破坏 |
| ControlKind 改为开放 union 破坏 Zod 校验 | z.union 保留旧枚举向后兼容 |
| i18n label 无翻译时显示空 | fallback 到原始字符串（永不为空） |
| Widget 迁移后属性面板异常 | generateControls 签名完全一致，机械替换 |
| 主题切换后 Widget 颜色不对 | ctx.theme 可选，不传时用默认色 |
