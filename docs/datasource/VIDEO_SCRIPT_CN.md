## 💡 录制小贴士

1.  **清理环境**: 录制前清空浏览器书签栏，隐藏掉无关的系统通知。
2.  **鼠标轨迹**: 操作时鼠标移动不要太快，点击重要按钮时可以稍微停顿一下。
3.  **局部放大**: 在填写 URL 或编写表达式时，后期剪辑可以将画面局部放大，方便观众看清代码。
4.  **准备数据**: 提前准备好测试用的 JSON 和 API 接口，不要在录制时现场敲太多字。

## 📝 准备物料

-   **Demo JSON**:
    ```json
    { "temperature": 26.5, "humidity": 60, "status": "Running" }
    ```
-   **测试 API**: 使用 `https://jsonplaceholder.typicode.com/todos/1` (如果需要公网测试)
-   **测试 WS**: 使用 `wss://echo.websocket.org` (如果可用) 或本地 Mock 服务
