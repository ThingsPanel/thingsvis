# 初始化看板

用户首次安装系统后，thingsvis系统会默认一个email是admin@thingsvis.io的用户。
然后用户接下来会从iot平台登录超管，首次登录这时候thingsvis也会自动创建一个用户。该超管用户需要默认一个超管首页看板。要初始化到数据库。


当用户登录租户管理员后，要给每个租户管理员默认一个租户首页看板。要初始化到数据库。



要默认看板必须先默认注册一个项目
sql示例：
```sql
INSERT INTO thingsvis.projects
```
